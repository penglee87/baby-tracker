# Baby Tracker (宝宝记录)

微信小程序：宝宝成长记录助手。支持多宝宝管理、日常行为记录（喂奶、睡眠、排泄等）及数据统计。

## 功能特性

- **多宝宝管理**：支持多个宝宝的档案切换，数据独立存储。
- **快捷记录**：首页提供一键记录功能，支持自定义按钮排序。
- **数据统计**：
  - 今日概览：实时统计当天的关键数据。
  - 历史趋势：最近 7 天的数据可视化展示。
  - 每日明细：详细查看每一天的记录流水。
- **数据同步**：
  - 本地优先：无网络环境下自动使用本地存储。
  - 云端支持：预留云开发接口，支持数据云端备份（需配置云开发环境）。

## 快速开始

1. 使用 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 导入本项目。
2. AppID 请使用测试号或自己的小程序 AppID。
3. 也可以直接修改 `project.config.json` 中的 `appid` 字段。
